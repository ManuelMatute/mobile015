# Reading Habit  
_App de hábitos de lectura_

**Reading Habit** es una aplicación móvil y web desarrollada con **Ionic + React + TypeScript** que ayuda a las personas a crear y mantener un hábito de lectura de forma simple, visual y motivadora.

La app permite descubrir libros, organizarlos en listas personales y hacer seguimiento del progreso de lectura día a día, transformando la lectura en una experiencia más estructurada y constante.

---

## Objetivo del proyecto

El objetivo de Reading Habit es facilitar que los usuarios:

- Formen un hábito de lectura sostenible  
- Visualicen su progreso de forma clara  
- Se mantengan motivados mediante rachas y metas  
- Descubran libros alineados a sus gustos  

Todo esto sin fricción ni complejidad innecesaria.

---

## Funcionalidades principales

### Exploración de libros
La aplicación se conecta a **OpenLibrary** para buscar libros por título, autor o tema.  
Los usuarios pueden explorar y seleccionar libros para agregarlos a su biblioteca personal.

### Recomendaciones personalizadas
La app genera **recomendaciones de libros** basadas en las preferencias definidas durante el onboarding, como:
- **Nivel de lectura**
- **Géneros favoritos**
- **Meta diaria**

Esto permite mostrar sugerencias más relevantes para cada usuario y mejorar la experiencia desde el inicio.

### Biblioteca personal
Cada usuario cuenta con tres listas:
- **Leyendo**
- **Por leer**
- **Terminados**

Desde aquí se puede gestionar fácilmente cada libro.

### Seguimiento de progreso
Para cada libro se puede:
- Indicar cuántas páginas se han leído  
- Incrementar o reducir páginas con botones  
- Guardar el progreso manualmente  
- Ver el porcentaje de avance y una barra visual de progreso  

### Sistema de racha
Cada vez que el usuario marca que leyó hoy, se actualiza su racha, fomentando la constancia y la creación de un hábito diario.

### Onboarding personalizado
Al iniciar por primera vez, el usuario define:
- Su nivel de lectura  
- Sus géneros favoritos  
- Su meta diaria de lectura  

Estos datos se usan para personalizar la experiencia.

### Perfil del usuario
En el perfil se muestra:
- Un resumen de libros leídos, en progreso y pendientes  
- La racha actual  
- Las preferencias definidas en el onboarding  
- Accesos rápidos para reiniciar la racha o rehacer el onboarding

## Tecnologías utilizadas

- **Ionic Framework**
- **React**
- **TypeScript**
- **Capacitor**
- **OpenLibrary API**
- **CSS personalizado y diseño responsive**

## Diseño y experiencia

La interfaz fue diseñada siguiendo principios de claridad visual, jerarquía de información y adaptación correcta a dispositivos móviles y web.  
Se utilizaron **safe areas**, layouts flexibles y estilos personalizados para asegurar una experiencia fluida en Android y navegador.

## Estado del proyecto

Este proyecto representa una versión funcional y estable de una aplicación de hábitos de lectura, lista para ser utilizada y ampliada con nuevas funcionalidades como estadísticas, recomendaciones avanzadas o sincronización en la nube.
